# DistributedRayTracer
Distributed ray tracer developen in c++

- flat shading
- multi light
- soft shadows
- mirrors
- dielectric
- depth of field
- glossy spheres
- trimesh texture mapping

## Dependencies
- OpenCV 2.7
- GLM
